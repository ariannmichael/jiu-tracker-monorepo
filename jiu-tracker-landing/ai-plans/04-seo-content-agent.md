# SEO / Content Agent

## Mission

Write conversion-focused Portuguese copy and metadata that support both search visibility and landing-page clarity.

## Inputs

- `ai-plans/00-master-brief.md`
- `ai-plans/01-ux-structure-agent.md`
- reference image tone and hierarchy

## Deliverables

1. Final headline and subheadline options.
2. Button copy for all CTA placements.
3. Section copy:
   - hero
   - feature proof cards
   - core features
   - final CTA
   - footer microcopy
4. Metadata:
   - page title
   - meta description
   - Open Graph title
   - Open Graph description
5. Recommended keyword themes for future expansion.

## Copy constraints

- Portuguese-first.
- Short, concrete, benefit-led language.
- Avoid inflated startup jargon.
- Use keywords naturally, for example:
  - app de Jiu-Jitsu
  - tracker de treinos
  - estatísticas de treino
  - técnicas de Jiu-Jitsu

## SEO guidance

- Keep the title specific and brand-aware.
- Keep the description readable, not stuffed.
- Use one `h1` and logical `h2`s.
- Make alt text descriptive of the app visuals.
- Prepare a future FAQ section if search growth becomes a priority.

## Suggested messaging direction

- Headline should combine progress tracking and Jiu-Jitsu specificity.
- Subheadline should mention training logs, statistics, and performance improvement.
- CTA copy should be action-oriented:
  - `Baixar app`
  - `Ver recursos`
  - `Comece a acompanhar seus treinos`

## Definition of done

- The page copy is believable, concise, and tailored to BJJ practitioners.
- Metadata is ready to replace the default Next.js values immediately.
